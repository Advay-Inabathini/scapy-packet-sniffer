Small chunks of data that are sent over a network, similar to how a large file might be broken down into smaller envelopes for mailing.

The unique address that identifies a device on a network, similar to a street address for a house.
Source IP: The IP address of the device that sent the data packet.
Destination IP: The IP address of the device intended to receive the data packet.

Transmission Control Protocol is a reliable protocol for sending data like a conversation. It ensures data arrives in order and without errors.

User Datagram Protocol is a faster but less reliable protocol for sending data like throwing a letter in a mailbox. It prioritizes speed over guaranteed delivery.
